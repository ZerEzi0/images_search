# Поиск изображений по текстовому описанию (PoC)

Проект представляет собой Proof of Concept (PoC) системы поиска референсных фотографий для профессионального фотохостинга **«Со Смыслом» (With Sense)**. 

Сервис позволяет пользователям находить изображения, вводя текстовое описание сцены. Система анализирует семантическую близость текста и визуального контента, выдавая наиболее релевантные результаты.

## Цель проекта
Разработать демонстрационную версию поискового движка, который:
1. Принимает текстовый запрос от пользователя.
2. Находит изображения с похожим смысловым содержанием.
3. Оценивает сходство текста и изображения (score от 0 до 1).
4. Соблюдает юридические ограничения (фильтрация контента с участием детей).
